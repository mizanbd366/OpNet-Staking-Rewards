# OpNet-Staking-Rewards
OpNet Staking Rewards is a decentralized DeFi application that allows users to stake their tokens and earn passive rewards over time.

### 🚀 Features Included:
- **Token Staking:** Securely deposit tokens into the contract.
- **Time-Based Rewards:** Automated reward calculation based on staking duration.
- **Full Withdrawal:** Users can claim both their principal and earned rewards.

### 🛠 Technical Specifications:
- **Smart Contract:** Solidity (^0.8.20)
- **Token Standard:** ERC-20
- **Security:** OpenZeppelin Library
